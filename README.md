# Webinar Feedback & Certificate Portal

A Next.js 15 web application for webinar participants to submit feedback and download their certificates.

## Features

- ✅ Responsive design (Mobile, iPad, Desktop)
- ✅ Feedback form with star rating system
- ✅ Certificate download after feedback submission
- ✅ Blue theme with subtle grid pattern and blur effects
- ✅ Lucide React icons
- ✅ Shadcn/ui components
- ✅ Support for 3 instructors display

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Design Features

- Light blue and white color scheme
- Subtle grid pattern overlay
- Blue blur effects for ambient background
- Responsive layout for all device sizes
- Certificate size: 29.7cm x 21cm (A4 Landscape)

## Tech Stack

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS
- Shadcn/ui
- Lucide React Icons
